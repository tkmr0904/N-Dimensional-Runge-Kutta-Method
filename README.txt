n次元のルンゲクッタ法(4次近似)  
  
"RK.cpp"の関数の依存関係は以下のようになっている。(上の関数を利用して下の関数を動作させる)  
  
dif_RKn.................関数値 y(x), y'(x), y''(x),... から、hだけ先の関数値 y(x+h), y'(x+h), y''(x+h),...   
    ┃                   を求める処理。  
    ┃  
    ┃           vf11, vf12, vf21, vf22..........dy(x)/dx, dy'(x)/dx, dy''(x)/dx,... と y(x), y'(x), y''(x),...   
    ┃           ┃                               の関係性を表現する関数の簡単な例。  
    ┃           ┃                               これらの関数をRKnの引数として与えることで微分方程式を解くことができる。  
    ┃           ┃                               関数の形は解きたい微分方程式に応じて変わる。
    ┃           ┃  
    ┃           ┃    
    ┗━━━━━━━━━━━RKn.............y(x_begin), y'(x_begin), y''(x_begin),...の値を引数として受け取る。  
                ┃               そこでdif_RKnを何度も利用することによって、離れたxにおける関数値  
                ┃               y(x_end), y'(x_end), y''(x_end),...   
                ┃               を求めることができる。  
                ┃  
                main............簡単な微分方程式を数値的に解き、程度の精度で計算が求まるのか解析解と比較している。  
                
また、"Vec.h"にはn次元ベクトルのクラスがテンプレートで定義されている。
